# SOTS Tag Audit (Add-only)

Goal: find missing GameplayTags references and propose add-only fixes.

1) Use MCP: `sots_run_devtool` to run your tag usage report (if available in your toolbox)
2) Summarize:
   - missing tags
   - which plugin(s)/file(s) reference them
3) Propose:
   - add-only entries to DefaultGameplayTags.ini
4) Do NOT remove/rename tags.
