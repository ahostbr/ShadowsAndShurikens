    Node: Input System Overview :
All Widgets That Inherit From This Widget Class Will Add IMC_UI_Controls Mapping Context As Well As Adding Themselves To The Player Controller "InputEnabledWidgetStack".
When The PC Recieves Input It Will Send It To The Top Of The Stack.  Via The BPI_CGF_User_Interface Events.
When A Widget Losses Focus Or is Collapsed/Hidden It Is Popped And The Stack Is Checked,
If Empty The IMC_UI Controls Mapping Is Removed. (EdGraphNode_Comment)
